# Интерактивное приложение для подготовки к коллоквиуму 2 по ЛАиГ

## Что внутри

- **Все 133 определений** с поиском по тексту, разнесены по разделам.
- **~60 теорем** с раскрывающимися блоками: формулировка -> идея -> полное доказательство -> подводные камни.
- **Пошаговые алгоритмы** с слайдером шагов:
  - Метод Якоби с угловыми минорами (раздел 3)
  - Метод наименьших квадратов (раздел 4)
  - Грам-Шмидт (раздел 4)
  - Спектральный анализ оператора (раздел 6)
  - SVD через спектр AᵀA (раздел 7)
- **Интерактивные геометрические визуализации** (Plotly, можно крутить мышью):
  - Сумма подпространств в R³ (раздел 1)
  - Линии уровня квадратичной формы с главными осями (раздел 3, 7)
  - Проекции вектора на прямую/плоскость в 2D и 3D (раздел 4)
  - Векторное произведение и параллелограмм в 3D (раздел 5)
  - Действие оператора на единичный круг с собственными векторами (раздел 6, 7)
  - Поворот в 2D с регулировкой угла слайдером (раздел 7)
- **Калькулятор расстояний** в R³ (раздел 5): точка-прямая, точка-плоскость,
  скрещивающиеся прямые.
- **Тренажёр**: генератор билетов в формате коллоквиума с таймером 10/25 минут.

## Установка и запуск

### Способ 1: через скрипт

**Windows.** Открыть `start.bat`. Откроется окно консоли, при первом
запуске установятся зависимости (~1-2 мин), после чего всё отобразится на
http://localhost:8501.

**Linux/Mac.** Запустить `./start.sh` в терминале.

### Способ 2: вручную

```bash
pip install -r requirements.txt
streamlit run app.py
```

### Создание ярлыка на рабочем столе (Windows)

1. ПКМ на `start.bat` -> «Создать ярлык».

### Создание `.desktop`-файла на Linux

Создать файл `~/.local/share/applications/laig.desktop`:

```ini
[Desktop Entry]
Type=Application
Name=ЛАиГ
Comment=Подготовка к коллоквиуму 2
Exec=/полный/путь/к/start.sh
Terminal=true
Icon=accessories-calculator
Categories=Education;
```

## Зависимости

- **streamlit** — UI-фреймворк.
- **numpy, scipy** — линейная алгебра.
- **plotly** — интерактивные графики 2D/3D с поворотом мышью.
- **pandas** — таблицы для отображения матриц.

## Как править

Все формулировки определений и теорем — в Python-файлах внутри `pages/`,
обычные строки.

Алгоритмы (Грам-Шмидт, Якоби, SVD) — в `laig/algorithms.py`. Каждый
возвращает результат и список объектов `Step` со своими описаниями.

Визуализации — в `laig/viz.py`, всё на Plotly.

## !

- При первом запуске первая страница может загружаться 2-3 секунды
  (Streamlit компилирует кэш). Дальше всё быстро.
- Plotly 3D-графики иногда тормозят в Firefox при сложных сценах. Chrome
  работает плавно.
- Длинные матрицы (>8×8) обрезаются, чтобы не сломать вёрстку. Если нужно
  больше — можно править в `max_rows` / `max_cols` в `laig/ui.py`.

## Чего нет

- Симметричный алгоритм Гаусса для квадратичных форм (метод Якоби покрывает
  большинство случаев).
- Проверка ввода на корректность во всех краевых случаях (например, нулевой
  направляющий вектор для прямой просто выдаст ошибку или предупреждение).
- Подсказки по доказательствам в режиме экзамена.

Успехов в подготовке!
